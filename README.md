# Blogmark

Marking blog site. Like bookmark, but for blog.

## stack

- next.js
  - v13.x
  - app directory
- prisma
- sqlite
- cloudflare r2
- react-query, react-hook-form, zod
- ui
  - tailwindcss
  - radix-ui
  - framer-motion
- assets
  - icon: [lucide.dev](https://lucide.dev)
